CodeDraw

CodeDraw is an easy-to-use drawing library where you use code to create pictures and animations. It is made for beginners that understand little about programming and makes it very simple to draw and animate various shapes and images to a canvas.

The source code, documentation and further examples can be found in the CodeDraw repository.

If you are interested in the detailed designed decision that went into this library please read my bachelor thesis on CodeDraw CodeDraw: A Graphics Library for Novice Programmers.

Download ParticleGravity.java

A particle Game where each particle moves towards the mouse. The closer the particle gets towards the mouse the faster it moves.

Game of Life

Download GameOfLife.java

An implementation of Conway's Game of Life. The rules that decide which cells are white or black can be found in the Conway's Game of Life Wikipedia article. This implementation also lets the user place black cells by clicking on a white cell and moving the mouse and place white cells by clicking on a black cell and moving the mouse.
